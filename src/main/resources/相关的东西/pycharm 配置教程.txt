pycharm 

ide：集成开发环境（一个软件，开发工具 ）
      -编辑器
	  -调试环境
	  -解释器

-下载：百度搜索Pycharm
下载Python  网址：百度搜索python。可以直接看python环境搭建|菜鸟教程
下载 ：点击相应的版本  选择 Windows x86-64 executable installer
添加系统环境变量
激活pycharm
脚本模板设置：File->Settings->Editor->File and Code Templates

##!/usr/bin/python3# -*-
# coding: utf-8 -*-
# @Time : ${DATE} ${TIME}
# @Author : ${USER}
# @Email : 972354178@qq.com
# @File : ${NAME}.py
# @Project : ${PROJECT_NAME}

# !/usr/bin/env python
# -*- coding:utf-8 -*-
# author:覃义雄

添加R语言解释器：Files->Settings->Languages & Frameworks->R Language

设置字体大小及类型：File->Settings->Editor->Font

在pycharm里安装第三方模块
File->Settings->Project:项目名->Project Interpreter->‘+’->搜索->Install Package

shift + Enter ：在光标的下一行新建一行

Tab：选中要缩进的代码，按Tab键进行整体缩进
